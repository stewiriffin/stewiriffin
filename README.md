## About Me

Backend-focused developer with 4+ years shipping production software. I build clean APIs in the Python ecosystem and mobile apps with Flutter and native Android (Kotlin, Jetpack Compose).

I care about readable code, sensible architecture, and products that hold up under real constraints — flaky networks, messy data, and users who never read the docs.

- **Now:** WhatsApp Business automation, Android utility and streaming apps
- **Learning:** Machine learning and system design
- **Shipped:** 2 apps on [Google Play](https://play.google.com/store/apps/developer?id=Rank+Apps) plus open-source tools on GitHub
